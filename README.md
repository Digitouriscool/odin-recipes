# Odin Recipes

## Description

Odin Recipes is a simple HTML project built as part of **The Odin Project Foundations curriculum**. The goal of this project is to practice fundamental HTML skills by creating a small recipe website. The site consists of a main index page that links to several individual recipe pages.

Each recipe page contains structured content including a description of the dish, a list of ingredients, and step-by-step instructions on how to prepare the meal. This project focuses on using semantic HTML elements and organizing content in a clear and readable format.

## Live Demo
The website is deployed using GitHUb Pages and can be accessed here:

https://digitouriscool.github.io/odin-recipes/

## Project Goals

This project helps reinforce the following HTML fundamentals:

* Creating and structuring web pages using HTML
* Using headings, paragraphs, and lists
* Linking between pages using anchor tags
* Organizing content in a logical layout
* Building a multi-page website structure

## Recipes Included

The website currently includes the following recipes:

* Jollof Rice
* Waakye
* Fufuo and Light Soup

Each recipe page contains:

* A short description of the dish
* An ingredients list
* Step-by-step cooking instructions
